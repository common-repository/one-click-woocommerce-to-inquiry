Change WooCommerce to Inquiry Module

== Description ==

If you want to use WooCommerce as Inquiry module, where your customers has ability only to send inquiry related to products. If you want to restrict your customers to purchase / order for any product then this plugin is for you.

== Installation == 
Simply install and activate this plugin. This plugin has not any admin side menu option.
After activating just visit to your Store / Website. Your website / store is now converted into Inquiry.
